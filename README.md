# 📝 To-Do List Web App

A simple To-Do List application built using **HTML**, **CSS (Tailwind CSS)**, and **JavaScript**. This app allows users to:

- Add new tasks
- Delete individual tasks
- View all tasks on the page

---

## 🚀 Features

- Clean and responsive UI using Tailwind CSS
- Dynamically add tasks using JavaScript
- Delete tasks instantly
- Centered layout with neat spacing

---

## 💻 Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (Vanilla)

---

## 📁 Project Structure

```

/to-do-list-project
│
├── index.html       # Main HTML file
├── style.css        # Optional custom CSS (if used)
├── script.js        # JavaScript logic
└── README.md        # Project information

````

## 🧑‍💻 How to Use

1. Clone or download the repository.
2. Make sure the following files are in the same folder:
   - `index.html`
   - `script.js`
   - `style.css` *(if you are using any custom styles)*
3. Open `index.html` in a browser.
4. Type your task and click "Add".
5. To remove a task, click the "Delete" button next to it.
